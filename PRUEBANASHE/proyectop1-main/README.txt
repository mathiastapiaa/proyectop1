Gestor de Citas Medicas
========================

Descripcion:
Este programa permite gestionar citas medicas de manera sencilla y segura. Incluye funciones para agendar, mostrar, buscar y cancelar citas, asi como realizar copias de seguridad y restaurarlas.

Caracteristicas principales:
- Agendar cita: Registrar una nueva cita medica para un paciente.
- Mostrar citas: Visualizar todas las citas programadas.
- Borrar cita: Cancelar una cita existente.
- Buscar por cedula: Consultar citas asociadas a una cedula de paciente.
- Realizar backup: Crear una copia de seguridad de las citas.
- Restaurar desde backup: Recuperar citas desde el archivo de backup.

Requisitos:
- Windows
- Ejecutable: GestorCitas.exe
- DLL necesaria: IngresoDatosDLL.dll

Instrucciones de uso:
1. Ejecute GestorCitas.exe.
2. Siga las instrucciones en pantalla para gestionar las citas.
3. No elimine el archivo IngresoDatosDLL.dll ni la carpeta 'data'.

Notas:
- El sistema no permite agendar citas en fines de semana ni feriados nacionales.
- Solo se permiten fechas validas y futuras para agendar citas.
- El backup protege sus datos ante perdidas o errores.
